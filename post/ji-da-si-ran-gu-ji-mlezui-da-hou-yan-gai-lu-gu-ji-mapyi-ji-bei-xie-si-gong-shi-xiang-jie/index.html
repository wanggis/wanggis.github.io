贝叶斯公式

p(θ∣x)=p(x∣θ)p(θ)p(x)\begin{aligned} p(\theta|x) = \frac{p(x|\theta)p(\theta)}{p(x)} \end{aligned} p(θ∣x)=p(x)p(x∣θ)p(θ)​​

  • xxx:观察得到的数据（结果）;
  • θ\thetaθ:决定数据分布的参数（原因）;
  • p(θ∣x)p(\theta|x)p(θ∣x):posterior 后验概率；
  • p(θ)p(\theta)p(θ):prior 先验概率；
  • p(x∣θ)p(x|\theta)p(x∣θ):likelihood 似然估计;
  • p(x)p(x)p(x):证据；

似然函数p(x∣θ)p(x|\theta)p(x∣θ)的理解

对于函数p(x∣θ)p(x|\theta)p(x∣θ)，输入有两个：xxx表示某一个具体的数据；θ\thetaθ表示模型的参数。

  • 概率： 如果θ\thetaθ是已知确定的，xxx是变量，这个函数就是概率函数(probability function)，它描述了对于不同的样本点xxx，其出现的概率是多少；
  • 统计： 如果xxx是已知确定的，θ\thetaθ是变量，这个函数就是似然函数(likelihood function)，它描述了对于不同的模型参数θ\thetaθ，出现xxx这个样本点的概率是多少；

极大似然估计(Maximum likelihood estimation, MLE)

MLE是利用已知的样本结果信息，反推最具有可能（最大概率）导致这些样本结果出现的模型参数值。换句话说，MLE提供了一种给定观察数据来评估模型参数的方法，即“模型已定，参数未知”。

MLE的重要假设是，所有的样本采样都是独立同分布的。

例子一：造币厂

假设造币厂生产某种硬币，现在我们拿到了一枚这种硬币，想试试硬币是否均匀。我们可以通过抛这枚硬币，正反面出现的概率(记为θ\thetaθ)来进行判断。
我们拿这枚硬币抛了10次，得到的数据(x0x_0x0​)是：反正正正正反正正正反。我们想求的正面概率θ\thetaθ是模型参数，而抛硬币模型我们可以假设为二项分布。
那么，出现实验结果x0x_0x0​ 的似然函数则为：

p(x0∣θ)=(1−θ)×θ×θ×θ×θ×(1−θ)×θ×θ×θ×(1−θ)=θ7(1−θ)3=f(θ)\begin{aligned} p(x_0|\theta) = (1−\theta)×\theta× \theta× \theta× \theta×(1−\theta)×\theta×\theta×\theta×(1−\theta) = \theta_7(1-\theta)^3 = f(\theta) \end{aligned} p(x0​∣θ)=(1−θ)×θ×θ×θ×θ×(1−θ)×θ×θ×θ×(1−θ)=θ7​(1−θ)3=f(θ)​

最大似然估计，则是要最大化这个函数。我们可以画出f(θ)f(\theta)f(θ)的图像：

可以看出，在θ=0.7\theta = 0.7θ=0.7时，似然函数取得最大值。这样，我们已经完成了对θ\thetaθ的最大似然估计。即，抛10次硬币，发现7次硬币正面向上，最大似然估计认为正面向上的概率是0.7。

例子二：男女学生身高问题

假设我们需要调查学校的男生和女生的身高分布。比较简单直接的方法就是采样。我们在校园里面随机独立抽取100个男生和100个女生，然后分别统计抽样得到100个男，女生的身高。假设他们的身高服从高斯分布，则男生和女生的分布的参数分别为θ1=[u1,σ12]\theta_1 = [u_1, \sigma_{1}^{2}]θ1​=[u1​,σ12​], θ2=[u2,σ22]\theta_2 = [u_2, \sigma_{2}^{2}]θ2​=[u2​,σ22​]。因此，我们的任务就是将θ1\theta_1θ1​和θ2\theta_2θ2​进行求解。
在学校的那么多男生中，我们独立地按照概率密度p(x∣θ)p(x|\theta)p(x∣θ)抽取了100个男生，组成了样本集X={x1,x2,x3,...,xn}X=\{x_1, x_2, x_3, ..., x_n\}X={x1​,x2​,x3​,...,xn​}。由于样本之间是互相独立的，所以同时抽到这100个男生的联合概率分布为：

(1)L(θ)=L(x1,...,x3∣θ)=∏i=1np(xi∣θ)\begin{aligned} L(\theta) = L(x_1, ..., x_3| \theta) = \prod_{i=1}^{n}p(x_i|\theta) \tag{1} \end{aligned} L(θ)=L(x1​,...,x3​∣θ)=i=1∏n​p(xi​∣θ)​(1)

上述的式子反映了，在概率密度函数的参数为θ\thetaθ时，得到XXX这组样本的概率。在这里，XXX是已知的，而θ\thetaθ是未知的。因此将该函数称为参数θ\thetaθ相对于样本集x的似然函数(likelihood function)，记为L(θ)L(\theta)L(θ)。
在学校那么多男生中，我们一抽就抽到这100个男生，而不是其他人，这表明在整个学校中，这100个人出现的概率最大（个人认为，似然函数遵循的一个原则是，发生观察到的即为最大）。所以，我们要做的就是找到一个参数θ\thetaθ，其对应的似然函数L(θ)L(\theta)L(θ)最大，也就是说抽到这100个男生（身高）的概率最大。这个就叫做θ\thetaθ的最大似然估计量，记为：

θ^=arg max⁡L(θ)\hat{\theta} = \argmax L(\theta) θ^=argmaxL(θ)

为了便于分解，我们通常还会加一个对数函数，将累乘变为累加。

(2)H(θ=log⁡L(θ)=log⁡∏i=1np(xi∣θ)=∑i=1nlog⁡p(xi∣θ)H(\theta = \log L(\theta) = \log \prod_{i=1}^{n}p(x_i|\theta) = \sum_{i = 1}^{n} \log p(x_i|\theta) \tag{2} H(θ=logL(θ)=logi=1∏n​p(xi​∣θ)=i=1∑n​logp(xi​∣θ)(2)

似然函数极大化，最直接的方法就是求导，然后让导数为0，借这个方程得到的θ\thetaθ就是了(当然，前提是函数L(θ)L(\theta)L(θ)连续可微)。

求解最大似然函数估计值的一般步骤：

  1. 写出似然函数；
  2. 对似然函数取对数，并整理；
  3. 求导数，令导数为0， 得到似然方程；
  4. 解似然方程，得到的参数即为所求。

最大后验概率估计(Maximum a posterior, MAP)

MLE的思路是求参数θ\thetaθ，使似然函数p(xi∣θ)p(x_i|\theta)p(xi​∣θ)最大；MAP则是想 θ\thetaθ，使得p(xi∣θ)p(θ)p(x_i | \theta) p(\theta)p(xi​∣θ)p(θ)最大。求得的θ\thetaθ不单单让似然函数大，θ\thetaθ自己出现的先验概率也得大。换句话说，MAP融入了要估计量的先验分布在其中（这有点像正则化加惩罚项的思想，不过正则化利用的是加法，而MAP利用的是乘法）。
通过上一节我们可以知道，MLE的优化公式为：

θ^MLE=arg max⁡θf(x∣θ)\begin{aligned} \hat{\theta}_{MLE} = \argmax _{\theta}f(x|\theta) \end{aligned} θ^MLE​=θargmax​f(x∣θ)​

现在，假设θ\thetaθ的先验分布为ggg。通过贝叶斯理论，对于θ的后验分布如下式所示：

f(θ∣x)=f(x∣θ)g(θ)∫θf(x∣θ′)g(θ′)dθ′\begin{aligned} f(\theta|x) = \frac{f(x|\theta)g(\theta)}{\int _{\theta}f(x|\theta')g(\theta')d\theta'} \end{aligned} f(θ∣x)=∫θ​f(x∣θ′)g(θ′)dθ′f(x∣θ)g(θ)​​

MAP的优化目标为:

θ^MAP=arg max⁡θf(x∣θ)g(θ)∫θf(x∣θ′)g(θ′)dθ′=arg max⁡θf(x∣θ)g(θ)\begin{aligned} \hat{\theta}_{MAP} = \argmax_{\theta} \frac{f(x|\theta)g(\theta)}{\int _{\theta}f(x|\theta')g(\theta')d\theta'} = \argmax_{\theta}f(x|\theta)g(\theta) \end{aligned} θ^MAP​=θargmax​∫θ​f(x∣θ′)g(θ′)dθ′f(x∣θ)g(θ)​=θargmax​f(x∣θ)g(θ)​

例子：饼干口味问题

假设有五个袋子，各袋中都有无限量的饼干(樱桃口味或柠檬口味)，已知五个袋子中两种口味的比例分别是：

  • 樱桃 100%
  • 樱桃 75% + 柠檬 25%
  • 樱桃 50% + 柠檬 50%
  • 樱桃 25% + 柠檬 75%
  • 柠檬 100%

如果只有如上所述条件，那问从同一个袋子中连续拿到2个柠檬饼干，那么这个袋子最有可能是上述五个的哪一个？
我们首先采用最大似然估计来解这个问题，写出似然函数。假设从袋子中能拿出柠檬饼干的概率为p(我们通过这个概率p来确定是从哪个袋子中拿出来的)，则似然函数可以写作

p(两个柠檬饼干∣袋子)=p2\begin{aligned} p(两个柠檬饼干|袋子)=p^2 \end{aligned} p(两个柠檬饼干∣袋子)=p2​

由于p的取值是一个离散值，即上面描述中的0, 0.25, 0.5, 0.75, 1。我们只需要评估一下这五个值哪个值使得似然函数最大即可，得到为袋子5。这里便是最大似然估计的结果。

上述最大似然估计有一个问题，就是没有考虑到模型本身的概率分布，下面我们扩展这个饼干的问题。
假设拿到袋子1或5的机率都是0.1，拿到2或4的机率都是0.2，拿到3的机率是0.4，那同样上述问题的答案呢？这个时候就变MAP了。写出我们的MAP函数:

θ^MAP=p2×g\begin{aligned} \hat{\theta}_{MAP} = p^2 × g \end{aligned} θ^MAP​=p2×g​

根据题意的描述可知，ppp的取值分别为0, 0.25, 0.5, 0.75, 1。ggg的取值分别为0.1, 0.2, 0.4,0.2, 0.1.分别计算出MAP函数的结果为：0, 0.0125, 0.125, 0.28125, 0.1.由上可知，通过MAP估计可得结果是从第四个袋子中取得的最高。
上述都是离散的变量，那么连续的变量呢？假设X={x1,x2,...,xn}X=\{x_1, x_2,...,x_n\}X={x1​,x2​,...,xn​}为独立同分布N(μ,σ2)N(\mu, \sigma^2)N(μ,σ2)，μ\muμ有一个先验的概率分布为N(μμ,σμ2)N(\mu_{\mu}, \sigma_{\mu}^2)N(μμ​,σμ2​)。那么我们想根据X={x1,x2,...,xn}X=\{x_1, x_2,...,x_n\}X={x1​,x2​,...,xn​}来找到μ\muμ的最大后验概率。根据前面的描述，写出MAP函数为：

μ^MAP=arg max⁡μf(x∣μ)g(μ)=arg max⁡12πσμexp⁡(−12(μ−μμσμ2))∏i=1n12πσμexp⁡(−12(xi−μσ2))\begin{aligned} & \hat{\mu}_{MAP} = \argmax_{\mu}f(x|\mu)g(\mu) \\ & = \argmax \frac{1}{\sqrt{2\pi}\sigma_{\mu}} \exp(-\frac{1}{2}({\frac{\mu- \mu_{\mu}}{\sigma_{\mu}}}^2)) \prod_{i = 1}^{n}\frac{1}{\sqrt{2\pi}\sigma_{\mu}} \exp(-\frac{1}{2}({\frac{x_i- \mu}{\sigma}}^2)) \end{aligned} ​μ^​MAP​=μargmax​f(x∣μ)g(μ)=argmax2π​σμ​1​exp(−21​(σμ​μ−μμ​​2))i=1∏n​2π​σμ​1​exp(−21​(σxi​−μ​2))​

为了简化优化过程，取对数，可得：

(μ−μμσμ)+∑i=1n(xi−μσ)\begin{aligned} (\frac{\mu- \mu_{\mu}}{\sigma_{\mu}}) + \sum_{i = 1}^{n}(\frac{x_i- \mu}{\sigma} \end{aligned}) (σμ​μ−μμ​​)+i=1∑n​(σxi​−μ​​)

求导解得：

μ^MAP=nσμ2nσ2μ+σ2(1n∑i=1nxi)+σ2nσ2μ+σ2μ\begin{aligned} & \hat{\mu}_{MAP} = \frac{n\sigma^2_{\mu}}{n\sigma^2{\mu} + \sigma^2 }(\frac{1}{n}\sum_{i = 1}^{n}x_i) + \frac{\sigma^2 }{n\sigma^2{\mu} + \sigma^2 } \mu \end{aligned} ​μ^​MAP​=nσ2μ+σ2nσμ2​​(n1​i=1∑n​xi​)+nσ2μ+σ2σ2​μ​

以上便是对于连续变量的MAP求解的过程。

在MAP中我们应注意的是：
MAP与MLE最大区别是MAP中加入了模型参数本身的概率分布，或者说。MLE中认为模型参数本身的概率的是均匀的，即该概率为一个固定值。
